Dice Probability Graphing Utility

A tool I created which can be found here. The tool can be used to graph the probability distributions of rolling n-sided dice, and is able to generate graphs for both total value and successes over a threshold. A target result can optionally be entered, and the tool will return the total probability of obtaining that target result or higher.
