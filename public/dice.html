Dice Probability Graphing Utility

This is a tool that can be used to visualize and compare the probability distributions of rolling any number of n-sided dice. Simply enter the number of dice and how many faces each die should have, and the tool will show the probability distributions for the dice rolls.

The Add Graph button will create a new set of inputs that can be assigned values. Remove Graph deletes the last input in the list. Plot Graphs will take all current inputs and graph them.

In the future the tool will be able to take a target number input for each graph, and represent the total probability of rolling above that value.

The equation for exact dice probabilities can be found here.

Due to difficulties with factorial scaling for large dice and sides values, any graph in which the number of dice times the number of sides is greater than 150 will use a normal probability distribution instead of the exact formula. This will produce almost completely accurate graphs for graphs in which the number of dice is large, but will be less accurate the lower the number of dice.

In the future edge cases like these will be checked and given more appropirate graphs.
